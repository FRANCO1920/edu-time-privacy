<h1>Política de Privacidad – TIME EDU</h1>

<p>
    **TIME EDU** es una aplicación de escritorio cuyo propósito es notificar al usuario sobre nuevos correos
    recibidos en su cuenta Gmail y mostrar un historial local de mensajes.  
    Para esto, solicita acceso <b>solo de lectura</b> mediante el permiso <b>gmail.readonly</b>.
</p>
